exp-semantic-skeleton
===

Expansive Semantic-UI skeleton plugin.

This provides:

 * Semantic-UI
 * JQuery
 * Configuration for Expansive plugins: less, js
 * Default layout 
 * Default partials 
 * Starter home page

### To install:

exp install semantic-skeleton

[https://github.com/embedthis/pak](https://github.com/embedthis/pak)
